# Рада (LLM Council) — Claude Code Skill

Скіл для Claude Code, що проганяє будь-яке рішення через **раду з 5 незалежних радників**. Кожен мислить із принципово іншого кута, потім вони анонімно рецензують одне одного, а голова синтезує фінальний вердикт.

Адаптація методології [LLM Council Андрея Карпаті](https://github.com/karpathy/llm-council): замість різних моделей — один Claude у п'яти різних лінзах мислення, які навмисно конфліктують між собою.

## Навіщо

Один AI дає одну відповідь — і ти не маєш як оцінити, наскільки вона повна, бо бачив лише один погляд. Рада дає об'єм: показує, **де радники сходяться** (сигнал високої впевненості), **де зчіплюються** (твоя реальна невизначеність) і **що робити першим**.

Інструмент універсальний — під будь-яке рішення, де помилитися дорого, не лише маркетинг.

## П'ять лінз

| Лінза | Роль мислення |
|---|---|
| **Скептик** | Шукає, де це зламається й що пропущено |
| **Археолог** | Зриває припущення: «що ми насправді вирішуємо?» |
| **Візіонер** | Бачить апсайд і масштаб, який інші проґавили |
| **Чужинець** | Нульовий контекст, свіже око, ловить «прокляття знання» |
| **Практик** | «Окей, а що робимо в понеділок зранку?» |

Три природні напруги: Скептик ↔ Візіонер (ризик/апсайд), Археолог ↔ Практик (переосмислити/зробити), Чужинець тримає всіх чесними.

## Як працює

1. **Рамка** — питання переформульовується нейтрально + підтягується контекст робочого простору.
2. **Скликання** — 5 лінз відповідають паралельно й незалежно.
3. **Peer-review** — відповіді анонімізуються, кожен рецензує колег наосліп.
4. **Синтез голови** — згода / конфлікт / сліпі плями / рекомендація / перший крок.
5. **Вивід** — фінальний вердикт прямо в чаті.

## Встановлення

Скопіюй теку `skills/council/` у свою теку скілів Claude Code:

```bash
cp -r skills/council ~/.claude/skills/
```

Або в конкретний проєкт: `.claude/skills/council/`.

## Використання

Тригери: `склич раду`, `стрес-тест це`, `перевір рішення`, `council this`, `pressure-test this`, а також м'які — `що обрати — X чи Y`, `не можу вирішити`.

```
склич раду: запускати воркшоп за 97 € чи курс за 497 €?
```

## Обмеження

В оригіналі Карпаті радники — це *різні моделі*. Тут усі 5 — той самий Claude у різних ролях, тож різноманіття дають промпти, а не різні движки. Для критичних рішень варто продублювати ключове питання в іншій моделі окремо.
